`Altium Libraries for Würth Elektronik Products`
=========
## Introduction
Welcome to the Würth Elektronik Altium Library repository.<br>
The libraries are created and maintained by the Würth Elektronik EDA Team.<br>
For any questions or suggestions, please feel free to contact the team at:  <libraries@we-online.com>

## Library Component Coverage
* Active Components <br>
_Digital Isolators, Power Modules (MagI³C Series), Radio Modules_

* Electromechanical Components <br>
_Assembly, Connectors, REDCUBE Terminals, Switches_

* Optoelectronic Components <br>
_7 Segments Display, IC LEDs, Infrared, Laser, LEDs, Optical Sensors, Optocoupler, Ultraviolet_

* Passive Components <br>
_Capacitors, Common Mode Chokes, EMC Shielding, ESD Protection, Ferrites for PCB Assembly, Filter Chokes, Surge Protection, PFC Chokes, Power Inductors, Transformers, Wireless Power Transmission, 
Quartz Crystals & Oscillators, Resistors, Antennas, Baluns, Filtered Connectors, RF Inductors, Signal Filters_
